# A* Search for Distance Between Two Cities

Uma   interface   entre   java   e   prolog   foi   criada   para   a   execução   das   buscas   de  
caminhos   entre   cidades   em   prolog,   através   da   interface   gráfica   do   java.   Para   executar  
o   aplicativo   (buscaCidadesProlog.jar),   o   código   fonte   em   prolog   (astarCidades.pl)   deve  
estar   no   mesmo   diretório   do   executável,   além   de   ser   necessária   a   instalação   do  
Swi­Prolog na máquina (sudo apt­get install swi­prolog swi­prolog­java).
